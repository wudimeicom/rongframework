单选按钮组

Rong_Html_Radios也是继承Rong_Html_Abstract。
它有两个属性：
public $options; //选项数组，值=>文本
public $checkedValue;//选中的值
两个方法：
public function setOptions( $options )//setter
public function setCheckedValue( $checkedValue )//setter
 
下面演示一下用法:
Filename: /test/html/html_radios.php
<?php 

/**
 * file encoding utf-8
 * 文件字符编码utf-8
 */
$PathToRongFramework = dirname(__FILE__)."/../../lib";

set_include_path(  "." . PATH_SEPARATOR . $PathToRongFramework .  PATH_SEPARATOR . get_include_path() );


require_once 'Rong/Html/Radios.php';

$radios = new Rong_Html_Radios( );
$radios->setOptions( array( 1=> "Jim", 2=>"LiLei", 3=> "HanMeiMei"));
$radios->setCheckedValue( 3 );
$radios->set("name", "friends" );

header("Content-Type: text/html; charset=utf-8");
echo $radios->toHtml();

输出：
<input  name="friends" type="radio" value="1"  />Jim  
<input  name="friends" type="radio" value="2"  />LiLei 
 <input  name="friends" type="radio" value="3" checked="checked"  />HanMeiMei
