For循环

For循环介绍

属姓名
类型说明
var变量循环数字的变量名，例如$index
from数值型循环初始值
to数值型循环结束值
step数值型每次增加的数值

为了少写wudimei模板引擎的初始化代码，我们把初始化代码放到load_wudimei()函数中。
我创建一个文件：application/includes/load.php , 它的内容是这样的：

<?php
/**
 * @return Rong_View_Wudimei Description
 */
function load_wudimei()
{
    require_once 'Rong/View/Wudimei.php';
    $wudimei = new Rong_View_Wudimei();
    $wudimei->compileDir = ROOT . "/data/compiled";

    $wudimei->viewsDirectory = ROOT . "/application/views";
    $wudimei->leftDelimiter = "{";
    $wudimei->rightDelimiter = "}";
    return $wudimei;
}
?>

接下来我们创建一个ForController以测试wudimei模板的循环。当然我们要加载load.php,在此之前我们先创建一个文件:application/controllers/test/wudimei/ForController.php
<?php
require_once ROOT . '/application/includes/load.php';
class ForController extends Rong_Controller
{
    public function __construct()
    {
        parent::__construct();
    }
    public function indexAction()
    {
        $wudimei = load_wudimei();
        $wudimei->display("test/wudimei/For/index.html");
    }
}
?>

我们还要创建一个模板文件:application/views/test/wudimei/For/index.html.
我们接着显示1到9这个数字。
<h3>simple loop</h3>
            {for var=$number from=1 to=9 step=1}
               {$number} <br />
            {/for}

我们也可以每隔一个数字显示一次，只要把step设为2就可以了。
<hr />
            {for var=$number from=1 to=9 step=2}
               {$number} <br />
            {/for}

这时访问http://127.0.0.9/index.php/test/wudimei/For 就可以看到结果了。

接下来我们输出一个数组中的元素,在ForController中添加入下代码：
<?php
$friends = array("Jim","HanMeimei","LiLei");
        $wudimei->assign("friends", $friends);

在.../For/index.html中加入显示$friends的代码：
<h3>My Friends</h3>
            {for var=$friendNumber from=0 to=$friends|count-1 step=1}
               {$friends.$friendNumber} <br />
            {/for}

这时访问http://127.0.0.9/index.php/test/wudimei/For 就可以看到结果了。
当然，你也可以倒过来显示，这时要换一下from和to,把step设为负数。
{for var=$friendNumber from=$friends|count - 1 to=0   step=-1}
               {$friends.$friendNumber} <br />
{/for}

我们再来输出一个二维数组,还是在ForController中添加代码：

<?php
$languages = array(
            array("name" => "java", "company" => "oracle"),
            array("name" => "c#", "company" => "microsoft" ),
            array("name" => "php" , "company" => "zend")
        );
        $wudimei->assign("languages", $languages );

还是在.../For/index.html中加入显示$languages的模板代码：
<h3>languages </h3>
            langeuage | company <br />
            {for var=$languageIndex from=0 to=$languages|count-1 step=1}
               {$languages.$languageIndex.name}  | {$languages.$languageIndex.company}<br />
            {/for}
