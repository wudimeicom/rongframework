if、else、elseif语句



在动作中添加一个$age的变量，它的值是16，显然未成年嘛！

<?php
//assing var msg to template
$wudimei->assign("age", 16 );
$wudimei->display("hello/if_else.html");

接下来我们在模板中判断是否未满18岁：

<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    </head>
    <body>
        <div>
            {if $age<18}
               Forbidden!
            {/if}
        </div>
    </body>
</html>

我们可以看到{if $age<18}表示如果年龄小于18就输出Forbidden!的字样，{if}的结尾是{/if}
但是如果年龄大于等于18我们就显示欢迎：
<div>
            {if $age<18}
               Forbidden!
            {else}
               Welcome!
            {/if}
        </div>

还有我们要判断$age是否小于等于0和大于150的情况,这是超级人类的年龄:

<div>
            {if $age<18}
               Forbidden!
            {elseif $age<=0 || $age>150}
               Super human!
            {else}
               Welcome!
            {/if}
        </div>


尝试一些运算符：
{if $man->age>=18 && $age>=18}
       two adult!
 {/if}
            
 {if $man->age <= $man->age+$age}
      the man is not so old!
{/if}
             
{if $man->age>=18||$age>=18}
       at lease one is adult.
{/if}
