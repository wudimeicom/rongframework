Foreach

foreach 的属性
属性名 类型 说明
from变量表达式
要循环输出的数组的变量名,例如$languages
item
变量表达式每次循环的数组元素的变量名,例如$lang
key
变量表达式
每次循环的数组元素的键名,例如$langKey
$wudimei.foreach.{数组变量名}.index
int
每次循环都会增一，从0开始。“{数组变量名}”可以是上面的"languages"变量,注意没有$

假如有这样的php:
Filename: /test/tpl_wudimei/foreach.php
<?php
/**
 * file encoding utf-8
 * 文件字符编码utf-8
 */
$PathToRongFramework = dirname(__FILE__)."/../../lib";

set_include_path(  "." . PATH_SEPARATOR . $PathToRongFramework .  PATH_SEPARATOR . get_include_path() );


require_once 'Rong/View/Wudimei.php';
$wudimei = new Rong_View_Wudimei();
$wudimei->compileDir = dirname(__FILE__) . "/templates/compiled";
$wudimei->viewsDirectory = dirname(__FILE__) .  "/templates";

$wudimei->leftDelimiter = "{";
$wudimei->rightDelimiter = "}";



//assing var msg to template
$wudimei->assign("msg", "Hello,World!");

$languages = array(
            array("name" => "java", "company" => "oracle"),
            array("name" => "c#", "company" => "microsoft" ),
            array("name" => "php" , "company" => "zend")
        );
$wudimei->assign("languages", $languages );
        


$wudimei->display("hello/foreach.html");


对应有这样的模板：
Filename: /test/tpl_wudimei/templates/hello/foreach.html
<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    </head>
    <body>
        <div>{$msg}</div>
        
        	<table border="1">
                <tr>
                    <th>index</th>
                    <th>key</th>
                    <th>name</th>
                    <th>company</th>
                </tr>
            {foreach from=$languages item=$lang key=$k}
                <tr>
                     <td>{$wudimei.foreach.languages.index}</td>
                     <td>{$k}</td>
                    <td>{$lang.name}</td>
                     <td>{$lang.company}</td>
                </tr>
            {/foreach}
            </table>
            
    </body>
</html>
