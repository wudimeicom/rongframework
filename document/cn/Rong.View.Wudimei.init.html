初始化Wudimei模版引擎

让我们建一个php文件。

控制器的文件路径是：/test/template_wudimei.php
首先我们要先载入Wudimei类，并创建一个实例：

<?php
require_once 'Rong/View/Wudimei.php';
$wudimei = new Rong_View_Wudimei();

接着我们要设置一下编译过的模板要放在哪里。
<?php
$wudimei->compileDir = dirname(__FILE__) . "/template_wudimei/compiled";

我们还要设置一下我们制作的模板放在哪里:
<?php
$wudimei->viewsDirectory = dirname(__FILE__) .  "/template_wudimei";

我们要告诉wudimei引擎界限符是什么，以致于它能正常编译模板：
<?php
$wudimei->leftDelimiter = "{";
$wudimei->rightDelimiter = "}";

好啦，这样我们的模板引擎已经初始化完毕了。

我们接下来测试一下模板引擎是否正常工作,我们设定一个$msg变量，它的值是“Hello,World!”，然后在“/test/template_wudimei/hello/index.html”中显示：

<?php
//assing var msg to template
$wudimei->assign("msg", "Hello,World!");

$wudimei->display("hello/index.html");

我们接下来新建一个模板文件"/test/template_wudimei/hello/index.html"

<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    </head>
    <body>
        <div>{$msg}</div>
    </body>
</html>

看到{$msg}这个变量了吗？它相当于输出“<?php echo $this->data["msg"]; ?>”。

接下来我们访问“/test/template_wudimei.php”，就可以看到"Hello,World!"字样了！
 

附上这个程序的源码：
文件名：/test/template_wudimei.php

 

<?php
/**
 * file encoding utf-8
 * 文件字符编码utf-8
 */
$PathToRongFramework = dirname(__FILE__)."/../../lib";

set_include_path(  "." . PATH_SEPARATOR . $PathToRongFramework .  PATH_SEPARATOR . get_include_path() );


require_once 'Rong/View/Wudimei.php';
$wudimei = new Rong_View_Wudimei();
$wudimei->compileDir = dirname(__FILE__) . "/template_wudimei/compiled";
$wudimei->viewsDirectory = dirname(__FILE__) .  "/template_wudimei";

$wudimei->leftDelimiter = "{";
$wudimei->rightDelimiter = "}";

//assing var msg to template
$wudimei->assign("msg", "Hello,World!");

$wudimei->display("hello/index.html");
