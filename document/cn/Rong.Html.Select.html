选择框

Rong_Html_Select也是继承Rong_Html_Abstract。
它有两个属性：
public $options; //选项数组，值=>文本
public $selectedValue;//选中的值
三个方法：
public function setOptions( $options )//setter
public function setSelectedValue( $selectedValue )//setter
 public function getOptions($row, $selectedValue)//取得选项的html
下面演示一下用法:
Filename: /test/html/html_select.php
<?php

/**
 * file encoding utf-8
 * 文件字符编码utf-8
 */
$PathToRongFramework = dirname(__FILE__)."/../../lib";

set_include_path(  "." . PATH_SEPARATOR . $PathToRongFramework .  PATH_SEPARATOR . get_include_path() );

require_once 'Rong/Html/Select.php';

$select = new Rong_Html_Select();
$select->set("name", "friends" );
$select->set("id", "id_friends");
$select->set("onchange","friends_onchange(this)");
$select->setOptions( array( 1=> "Jim", 2=>"LiLei", 3=> "HanMeiMei"));
$select->setSelectedValue( 2 );

header("Content-Type: text/html; charset=utf-8");
echo $select->toHtml();




输出：
<select  name="friends" id="id_friends" onchange="friends_onchange(this)" >
<option value="1" >Jim</option>
<option value="2"  selected="selected" >LiLei</option>
<option value="3" >HanMeiMei</option>
</select>
