Html控件

Html控件
我们来看一下抽像类Rong_Html_Abstract
它有以下两个属性：
public $tagName ; //这是html元素的标记名，例如input、div等
public $attributes = array();//这是标记的属性数组。
它有以下几个方法：
public function set($name, $value)//设置名称为$name,值为$value的属性。
public function remove( $name )//把名称为$name的属性删除
public function attributesToHtml()//把属性数组转换成字符串

所有html控件都继承Rong_Html_Abstract类。
