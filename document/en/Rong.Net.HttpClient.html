Http Client


the http client is using for request the remote web page,and can be post data and file.

To use it ,please include the class and create an instance.
<?php
/**
 * file encoding utf-8
 * 文件字符编码utf-8
 */
$PathToRongFramework = dirname(__FILE__)."/../../lib";

set_include_path(  "." . PATH_SEPARATOR . $PathToRongFramework .  PATH_SEPARATOR . get_include_path() );


require_once 'Rong/Net/HttpClient.php';

$client = new Rong_Net_HttpClient();

tell which directory to store the cookie.
<?php
$client->setCookieDir( dirname(__FILE__) . "/data/cookies" );

assumes that the page we want to request is net_demo_page.php
<?php
$url = "http://127.0.0.9/test/net_demo_page.php";

we request the web page by "GET" method.
<?php

//1.GET
echo "--------------request method GET-------------<br />";
$client->request( $url . "?name=YangQingRong" , "GET");
$response = $client->getContent();

echo "server response:". $response;

request() arguments:
$url string url
$method string GET or POST
$postArray array $key=>$value array,for post to the web page by "POST" method.

let 's post data and file:
<?php


//2.POST
echo "<br />--------------request method POST-------------<br />";
$client->addFile("photo", dirname(__FILE__) . "/data/Yqr.jpg" );
$postData = array("title"=>"hello,world","content"=>"good good study,day day up" );
$client->request( $url . "?name=YangQingRong" , "POST",$postData);
$response = $client->getContent();

echo "server response:". $response;

the method getContent() return the page content.
addFile($name,$path) is for adding file to request package.$name is the attribute of <input type="file" name="photo"/>
