initialize Wudimei Template Engine

Now,let us create a php file,
The path of this file is "/test/template_wudimei.php"
First of all,we need require the class of Wudimei template engine and create an instance for it.

<?php
require_once 'Rong/View/Wudimei.php';
$wudimei = new Rong_View_Wudimei();

second,we set the path where does the compiled templates located.

<?php
$wudimei->viewsDirectory = dirname(__FILE__) .  "/template_wudimei";

let us set path of templates we make.
<?php
$wudimei->viewsDirectory = dirname(__FILE__) .  "/template_wudimei";


we have to tell Wudimei Engine what is the delimiters.let wudimei compile templates correctly.

<?php
$wudimei->leftDelimiter = "{";
$wudimei->rightDelimiter = "}";

Ok,our template engine has been initialized!

now,we will test the wudimei engine if it work or not.we assign a variable call "$msg" with the value"Hello,World".and display it with template "/test/template_wudimei/hello/index.html"

<?php
//assing var msg to template
$wudimei->assign("msg", "Hello,World!");

$wudimei->display("hello/index.html");

and the we create a template mentioned above"/test/template_wudimei/hello/index.html"

<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    </head>
    <body>
        <div>{$msg}</div>
    </body>
</html>

do you see the variable "{$msg}"?it equals the code :<?php echo $this->data["msg"]; ?>.

ok,let us visit the url:/test/template_wudimei.php,you will see "Hello,World!"

the source code :
filename:/test/template_wudimei.php

 

<?php
/**
 * file encoding utf-8
 * 文件字符编码utf-8
 */
$PathToRongFramework = dirname(__FILE__)."/../../lib";

set_include_path(  "." . PATH_SEPARATOR . $PathToRongFramework .  PATH_SEPARATOR . get_include_path() );


require_once 'Rong/View/Wudimei.php';
$wudimei = new Rong_View_Wudimei();
$wudimei->compileDir = dirname(__FILE__) . "/template_wudimei/compiled";
$wudimei->viewsDirectory = dirname(__FILE__) .  "/template_wudimei";

$wudimei->leftDelimiter = "{";
$wudimei->rightDelimiter = "}";

//assing var msg to template
$wudimei->assign("msg", "Hello,World!");

$wudimei->display("hello/index.html");
