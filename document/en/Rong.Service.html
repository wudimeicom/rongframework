remote calling service

   Assumes that you need to call API from other server, Rong Framework provides calss Rong_Service_Server can be very good. It provides transport encryption, you can call the functions and methods available on the remote server,  Just like in the same local host calling.

    Assumes that the remote service http://127.0.0.9/test/service_server.php
Local call: http://127.0.0.1/test/service_client.php
For convenience, I put them in the same folder, you can move to the server you want.
