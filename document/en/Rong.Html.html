Rong Html Controls

Rong Html Controls
class Rong_Html_Abstract
the class has to attributes:
public $tagName ; //the tag name of html element,eg:input,div
public $attributes = array();//the attributes array of element

It has several methods:
public function set($name, $value)//setting an attribute with name $name,and assign value $value to it
public function remove( $name )//remove an attribute which name is $name
public function attributesToHtml()//convert attributes to html array

all html controls extends class Rong_Html_Abstract
