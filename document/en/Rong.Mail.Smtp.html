SMTP

Properties of class Rong_Mail_SendMail
Property Name type des
to array receipents
from string email of sender
subject string subject
content string content of email
ssl boolean use ssl or not.default is false
debug boolean output debug infomation or not
host string host name of smtp. eg:smtp.wudimei.com
port integer the port of smtp host,eg:25
username string user name of smtp
password string password of smtp
Methods of class Rong_Mail_SendMail
method name parameters return des
connect() none void connect to smtp host
login() none void login to the server
send() none void send email
quit() none void quit the send mode
close() void void close the smtp connection

Filename: /test/mail/mail_smtp_simple.php
<?php

//------------send second email start 发第二封邮件开始------

$smtp->send();
