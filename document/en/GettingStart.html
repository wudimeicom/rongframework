Getting start

This text is about getting start with RongFrameworkã€‚
1.Create the directories and files

+-- .httaccess   #for enable url rewirte
+-- index.php
+-- application
      +-- controllers  #controllers folder
      +-- models #database models folder
      +-- views #html views folder
 +-- lib
     +-- Rong #rong framework lib folder

2.Create a controller
create a file : /application/controllers/IndexController.php , and then fill in the code bellow:

 

<?php 
class IndexController extends Rong_Controller 
{
    public function indexAction()
    {
        echo "this is the index action of Index controller";
    }
    public function testAction()
    {
        echo "This is test action";
    }
}

3.create /index.php

 

<?php define( "ROOT" , dirname( __FILE__ ) );
set_include_path( "." . PATH_SEPARATOR . ROOT."/lib". PATH_SEPARATOR. get_include_path() );
require_once "Rong/Controller/Guide.php";
$guide = new Rong_Controller_Guide();
$guide->setControllerDir( ROOT . "/application/controllers"); //set the path of controllers
$guide->start();

 

4.Does your server enable url rewrite?
a) no,i don't need it
please visit your Index controller test action like this:

 

http://localhost/index.php?do=/Index/test #visit rule:index.php?do= /Controller/action or index.php?do=/subControllerFolderName/Controller/action


b)Yes,i need url rewrite
if you want to enable url-rewrite,please open the httpd.conf and find out the text bellow:

#LoadModule rewrite_module modules/mod_rewrite.so

please remove the "#" at the front of the line , it's a comment char.


Options Indexes FollowSymLinks MultiViews ExecCGI
AllowOverride All
Order allow,deny
Allow from all

 

Make sure the Options contains the "FollowSymLinks" section, if there have no ,please add one! and Make sure the value of "AllowOverride" is "All". save the changes and restart apache.

create a file ".htaccess" and place it to the ROOT folder of you application.


RewriteEngine On
RewriteBase /
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php/$1 [L]


ErrorDocument 404 /index.php

ok,now you visit the url bellow,and then the test action output the result .

http://localhost/Index/test
