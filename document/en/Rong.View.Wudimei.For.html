For Loop Introduction

Attribute Name
typedescription
varvariablethe variable name of "for" loop.eg:$index
fromnumberthe initial value of variable
tonumberthe end value of variable
stepnumbervalue plused each loop

For writing less initial code for wudimei template engine,we put the initial into function load_wudimei().
Let's create a file application/includes/load.php with code below:

<?php
/**
 * @return Rong_View_Wudimei Description
 */
function load_wudimei()
{
    require_once 'Rong/View/Wudimei.php';
    $wudimei = new Rong_View_Wudimei();
    $wudimei->compileDir = ROOT . "/data/compiled";

    $wudimei->viewsDirectory = ROOT . "/application/views";
    $wudimei->leftDelimiter = "{";
    $wudimei->rightDelimiter = "}";
    return $wudimei;
}
?>

now,let's create a controller ForController to test the loop of wudime template engine.
File:application/controllers/test/wudimei/ForController.php
<?php
require_once ROOT . '/application/includes/load.php';
class ForController extends Rong_Controller
{
    public function __construct()
    {
        parent::__construct();
    }
    public function indexAction()
    {
        $wudimei = load_wudimei();
        $wudimei->display("test/wudimei/For/index.html");
    }
}
?>

let's create a template file:application/views/test/wudimei/For/index.html.
the code below is for displaying from 1 to 9.
<h3>simple loop</h3>
            {for var=$number from=1 to=8 step=1}
               {$number} <br />
            {/for}

by setting the attribute to 2,we can see:1 3 5 7 9
<hr />
            {for var=$number from=1 to=9 step=2}
               {$number} <br />
            {/for}

visit http://127.0.0.9/index.php/test/wudimei/For to see the result


let's display the elements of an array,add an array $friends into ForController which we have crated.

<?php
$friends = array("Jim","HanMeimei","LiLei");
        $wudimei->assign("friends", $friends);

add code into .../For/index.html to display $friends
<h3>My Friends</h3>
            {for var=$friendNumber from=0 to=$friends|count-1 step=1}
               {$friends.$friendNumber} <br />
            {/for}

visit url http://127.0.0.9/index.php/test/wudimei/For to see the result.
we also could show it reversely,add minus to "step"'s value,like this way:
{for var=$friendNumber from=$friends|count - 1 to=0   step=-1}
               {$friends.$friendNumber} <br />
{/for}


let's output a two-dimensional array,we declare an array $language with given value and assign it to wudimei template.

<?php
$languages = array(
            array("name" => "java", "company" => "oracle"),
            array("name" => "c#", "company" => "microsoft" ),
            array("name" => "php" , "company" => "zend")
        );
        $wudimei->assign("languages", $languages );

add code into .../For/index.html to display $languages:
<h3>languages </h3>
            langeuage | company <br />
            {for var=$languageIndex from=0 to=$languages|count -1  step=1}
               {$languages.$languageIndex.name}  | {$languages.$languageIndex.company}<br />
            {/for}
